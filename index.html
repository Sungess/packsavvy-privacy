1. 개인정보보호정책 개요

PackSavvy("당사", "앱")는 사용자의 개인정보보호를 중요하게 생각합니다. 본 개인정보보호정책은 PackSavvy 모바일 애플리케이션 사용 시 개인정보 처리 방침을 설명합니다.

핵심 원칙: PackSavvy는 모든 데이터를 사용자 기기에 로컬로 저장하며, 외부 서버로 개인정보를 전송하지 않습니다.

2. 수집하는 정보

2.1 직접 제공 정보

  • 여행 정보: 여행명, 목적지, 날짜, 목적
  • 체크리스트 데이터: 준비물 목록, 포장 상태, 가방 정보
  • 사용자 설정: 언어 설정, 테마 설정
  • 여행 서류: 여권, 비자, 항공권 등의 이미지 또는 PDF 파일
  • 서류 이미지: 카메라로 촬영하거나 업로드한 서류 파일

2.2 자동 수집 정보

  • 기기 정보: 운영체제 버전, 앱 버전
  • 사용 패턴: 기능 사용 빈도 (기기 내부에서만 처리)

3. 정보 사용 목적

수집된 정보는 다음 목적으로만 사용됩니다:

  • 여행 준비물 체크리스트 생성 및 관리
  • 여행 서류의 안전한 보관 및 관리
  • 오프라인 서류 접근 서비스 제공
  • 사용자 맞춤형 앱 설정 제공
  • 앱 기능 개선 및 최적화
  • 기술적 문제 해결 및 고객 지원

4. 정보 저장 및 보안

로컬 저장: 모든 개인정보는 사용자의 기기에만 저장되며, 외부 서버나 클라우드로 전송되지 않습니다.

4.1 보안 조치

  • 기기 내 암호화된 로컬 저장소 사용
  • 모든 서류 파일은 기기 내부에 암호화되어 저장
  • 서류 데이터는 서버로 전송되지 않으며, 외부 접근이 차단
  • 앱 삭제 시 모든 데이터 및 서류 완전 삭제
  • 외부 네트워크 전송 없음

4.2 데이터 보관 기간

데이터 및 서류는 사용자가 앱을 삭제하거나 직접 데이터를 삭제할 때까지 기기에 보관됩니다.

5. 제3자 공유

정보 공유 없음: PackSavvy는 사용자의 개인정보를 제3자와 공유, 판매, 임대하지 않습니다.

다음의 경우에만 예외적으로 정보 공개가 있을 수 있습니다:

  • 법적 요구 또는 법원 명령이 있는 경우
  • 사용자의 명시적 동의가 있는 경우
  • 앱의 안전성 및 보안 위협 대응을 위해 필요한 경우

6. 사용자 권리

사용자는 다음과 같은 권리를 가집니다:

  • 접근권: 앱 내에서 본인의 모든 데이터 확인 가능
  • 수정권: 언제든지 여행 정보 및 체크리스트 수정 가능
  • 삭제권: 개별 데이터 삭제 또는 앱 삭제를 통한 전체 데이터 삭제
  • 이동권: 필요시 데이터 내보내기 기능 제공

7. 아동 개인정보보호

PackSavvy는 13세 미만 아동으로부터 의도적으로 개인정보를 수집하지 않습니다. 만약 13세 미만 아동의 개인정보가 수집되었음을 발견하는 경우, 즉시 해당 정보를 삭제하겠습니다.

8. 국제적 데이터 전송

PackSavvy는 로컬 저장만 사용하므로 국경을 넘나드는 데이터 전송이 발생하지 않습니다.

9. 정책 변경

개인정보보호정책이 변경되는 경우:

  • 앱 업데이트를 통해 사용자에게 알림
  • 중요한 변경사항의 경우 별도 공지
  • 변경된 정책은 공지 후 7일 후부터 적용
